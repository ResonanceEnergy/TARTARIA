4. Export as FBX with correct scale (1 unit = 1 meter)

### **Crystal Shader Customization:**
```csharp
// In CustomShaderApplicator.cs, add AetherCrystal variant
Material crystalMat = CreateMaterial("Tartaria/AetherCrystal", m => {
    m.SetColor("_TintColor", new Color(1f, 0.8f, 0.4f, 0.6f)); // Amber
    m.SetFloat("_Translucency", 0.7f);
    m.SetFloat("_RimPower", 3.0f);
    m.SetColor("_RimColor", new Color(0.8f, 0.9f, 1f, 1f)); // Blue-white rim
});
```

### **Aurora Skybox Animation:**
```csharp
// In SkyboxController.cs
public void AnimateAuroraIntensity(float gridStrength) {
    // Aurora intensity tied to global Aether grid strength
    RenderSettings.skybox.SetFloat("_AuroraIntensity", gridStrength);
    RenderSettings.skybox.SetColor("_AuroraColor", 
        Color.Lerp(Color.green, Color.magenta, gridStrength));
}
```
