## 📖 PART 10: LICENSE COMPLIANCE GUIDE

### CC0 (Public Domain) — NO ATTRIBUTION REQUIRED
**Use freely:** Commercial, personal, modify, redistribute  
**No requirements:** Zero restrictions, 100% freedom  
**Examples:** Polyhaven, KayKit, Quaternius, Tallbeard

### CC-BY (Creative Commons Attribution) — ATTRIBUTION REQUIRED
**Use freely:** Commercial, personal, modify, redistribute  
**Requirement:** Credit the author in game credits or README  
**Format:** "Asset Name by Author Name (CC-BY)"  
**Examples:** Many OpenGameArt assets

### OFL (SIL Open Font License) — FONT-SPECIFIC
**Use freely:** Commercial, personal, modify, redistribute  
**Requirement:** Include OFL.txt in distribution (for fonts)  
**Cannot:** Sell font standalone (must be part of larger work)  
**Example:** Monogram font

### Unity EULA — UNITY ASSET STORE
**Use freely:** Commercial/personal Unity projects only  
**Cannot:** Extract and use in non-Unity engines  
**Cannot:** Redistribute raw assets  
**Examples:** Human Basic Motions, PrimeTween

### Mixamo License — ADOBE-SPECIFIC
**Use freely:** Commercial, personal, modify  
**Cannot:** Redistribute raw FBX files  
**Can:** Include baked animations in final game  
**Example:** All Mixamo characters

## 🚀 PART 11: INTEGRATION WORKFLOW

### Step 1: Download All Assets (1-2 days)

```powershell
# Create directory structure
New-Item -ItemType Directory -Path "Assets\_Project\Models\Characters\Mixamo"
New-Item -ItemType Directory -Path "Assets\_Project\Textures\Polyhaven"
New-Item -ItemType Directory -Path "Assets\_Project\Audio\Music"
New-Item -ItemType Directory -Path "Assets\_Project\VFX\Particles"
New-Item -ItemType Directory -Path "Assets\_Project\Environments\OpenGameArt"
```

### Step 2: Import to Unity (2-4 hours)

```
1. Drag all FBX files to Assets\_Project\Models\Characters\
2. Drag all textures to Assets\_Project\Textures\
3. Drag all audio to Assets\_Project\Audio\
4. Drag all VFX prefabs to Assets\_Project\VFX\
5. Run .\apply-visual-upgrades.ps1 -P0Only (processes Mixamo FBX)
```

### Step 3: Replace Primitives (4-8 hours)

**Characters:**
- Open `Assets\_Project\Scripts\Integration\PlayerSpawner.cs`
- Change `playerPrefab` from null to `Player` prefab (created by FBXImportWizard)
- Repeat for BuildingSpawner, EchohavenContentSpawner

**Buildings:**
- Open `Assets\_Project\Scripts\Integration\BuildingSpawner.cs`
- Replace `CreateGreyboxBuilding()` with `LoadModularBuilding()`
- Wire KayKit modular pieces into Star Dome, Harmonic Fountain, Crystal Spire

**Terrain:**
- Apply Polyhaven textures to terrain via Terrain Inspector
- Replace skybox with Polyhaven HDRI (Lighting → Environment → Skybox Material)

### Step 4: Test & Validate (1-2 hours)

```powershell
# Run build pipeline
.\tartaria-play.ps1 -BatchOnly

# Check for errors
# Expected: 25/25 phases pass, 0 errors
# Visual: Characters visible, buildings modular, skybox realistic
```
