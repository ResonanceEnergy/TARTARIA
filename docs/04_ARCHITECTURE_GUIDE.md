## 1. Design Philosophy

Tartarian architecture was not decorative — it was **functional resonance technology**. Every curve, dome, spire, and moat served an energy purpose. The game's building system reflects this by making aesthetics and mechanics inseparable:

**Core Principle:** *The more beautiful a structure, the more powerful it is. Golden-ratio compliance isn't a cosmetic option — it's the energy multiplier.*

- Buildings that look wrong (asymmetric, non-golden-ratio) produce less Aether
- Buildings that look *right* produce cascading energy bonuses
- Players learn sacred geometry by feel — the system rewards visual harmony
- Even casual builders produce stunning structures because the snap system guides them

## 2. Architecture Archetypes

### 2.1 Domes & Onion Domes

**Function:** Resonance amplification chambers — collect and focus Aether from antenna spires

| Property | Value |
|---|---|
| **Shape** | Hemisphere or onion-bulb (ogee arch profile) |
| **Ideal Material** | Copper-clad (exterior), crystal-inlaid (interior) |
| **Resonance Role** | Amplifier — captures spire energy, focuses it downward |
| **Aether Band** | 3-Band (standard), 6-Band (with crystal lining), 9-Band (with mercury core) |
| **Size Range** | 5m (chapel) to 50m (grand cathedral) |
| **Visual Tell** | Glows from apex; intensity = RS level |

**Dome Physics:**
- Interior surface creates standing waves that amplify input frequency
- Onion domes have tighter focus (higher RS per unit area) but smaller coverage
- Half-domes (apse shape) redirect energy horizontally — used for connecting corridors

**Building Rules:**
- Must sit atop walls with minimum 2:1 height-to-width ratio
- Spire must crown the dome (no flat-top domes)
- Interior must be open (no internal floors cutting the resonance chamber)

### 2.2 Antenna Spires & Finials

**Function:** Aether receivers — pull atmospheric energy into the dome system

| Property | Value |
|---|---|
| **Shape** | Tapered cone, needle, or ornate finial |
| **Ideal Material** | Mercury-gilded copper (conductive + responsive) |
| **Resonance Role** | Collector — bridge between atmospheric Aether and structural system |
| **Height Bonus** | +5 RS per 10m above dome apex (capped at dome diameter) |
| **Visual Tell** | Sparks climb the spire at night; arcs during stormy weather |

**Spire Variants:**

| Type | Shape | Special Property |
|---|---|---|
| **Needle Spire** | Simple cone | Highest raw collection rate |
| **Mercury Orb Spire** | Sphere atop needle | Stores excess energy (battery function) |
| **Finial Spire** | Ornate (crescents, stars, crosses) | Shape resonance bonus (sacred geometry multiplier) |
| **Lightning Rod** | Metal rod with crystal tip | Converts storm energy to 6-band |
| **Frequency Fork** | Tuning-fork shape | Precise frequency selection |

**Placement Rules:**
- Always atop a dome (never freestanding)
- Taller = better, but structural ratio must be golden (1:1.618 base-to-height)
- Multiple spires per dome allowed (cathedral rooflines) — each adds diminishing returns

### 2.3 Star Forts

**Function:** Grounding nodes — anchor the Aether grid to planetary ley lines

| Property | Value |
|---|---|
| **Shape** | Geometric star polygon (5–12 points) |
| **Ideal Material** | Granite base, copper-water moat |
| **Resonance Role** | Grounder — stabilizes the ley-line grid, prevents Aether storms |
| **Geometry Bonus** | Higher point count = more stable routing |
| **Visual Tell** | Moats glow with conductive water; bastions hum at night |

**Star Fort Mechanics:**

| Points | Name | Grid Routing | Defense Rating |
|---|---|---|---|
| 5 | Pentagonal | 2 connections | Standard |
| 6 | Hexagonal | 3 connections | Enhanced |
| 8 | Octagonal | 4 connections | Superior |
| 12 | Dodecagonal | 6 connections | Legendary |

**Moat System:**
- Moats must be filled with **conductive pure water** (from fountain network)
- Deeper moats = stronger grounding
- Moats can be heated (thermal Aether) to create defensive mist barriers
- Connected moats between forts create routing highways

**Bastion Rules:**
- Each point of the star is a bastion
- Bastions mount defensive resonance cannons (combat)
- Must be symmetrical — asymmetric bastions reduce RS by 15%

## 3. Sacred Geometry Snap System

### 3.1 How Snap Works

When placing any building element, ghost-lines appear showing:

| Guide | Color | Meaning |
|---|---|---|
| **Golden Spiral** | Gold | Element aligns with φ (1.618) ratio |
| **Ley Line Vector** | Blue-white | Element sits on ley-line path |
| **Harmonic Circle** | Green | Element is within resonance range of neighbors |
| **Dissonance Warning** | Red pulse | Element conflicts with nearby geometry |

**Auto-Snap Behavior:**
- Elements within 5% of golden-ratio alignment auto-snap to perfect position
- Snap feels magnetic — gentle pull, satisfying click
- Players can override snap (freeplace mode) but lose geometry bonus
- Snap points are invisible by default; toggled into "Blueprint Mode" for precision builders

### 3.2 Geometric Templates

Pre-designed templates for common structures:

| Template | Shape | Use Case |
|---|---|---|
| **Vesica Piscis** | Two overlapping circles | Doorways, windows, junction points |
| **Seed of Life** | 7 overlapping circles | Foundation layouts |
| **Flower of Life** | 19 circles | Sacred floor patterns (RS boost) |
| **Metatron's Cube** | 13 circles + lines | Advanced dome vaulting |
| **Sri Yantra** | 9 interlocking triangles | Temple sanctum layouts |
| **Golden Rectangle** | φ-ratio rectangle | Wall panels, floor tiles |

Templates appear as translucent overlays that players can scale and rotate. Placing structures that match the template yields a **Geometry Compliance Bonus** (5–25 RS).

### 3.3 Fractal Scaling

Tartarian architecture is fractal — patterns at small scale repeat at large scale:

- Window tracery patterns mirror dome rib patterns
- Column spacing mirrors city block spacing
- Spire proportions mirror obelisk proportions

**Mechanic:** If a player builds a district where the macro-layout (block arrangement) matches the micro-layout (room arrangement) of their buildings, they earn a **Fractal Resonance Bonus** (+15 RS, zone-wide).

## 4. Golden Ratio Enforcement

### 4.1 The φ Grid

Every buildable surface is overlaid with an invisible φ-grid:
- Grid spacing follows the Fibonacci sequence (1, 1, 2, 3, 5, 8, 13...)
- Major structural elements (columns, arches, domes) must align with primary grid points
- Secondary elements (windows, buttresses) align with secondary grid points
- Decorative elements (finials, rosettes) can freeplace within grid cells

### 4.2 Proportion Checks

| Relationship | Golden Ratio | Tolerance | RS Penalty if Violated |
|---|---|---|---|
| Wall height : Wall width | 1.618 : 1 | ±5% | -10 RS |
| Dome height : Dome diameter | 0.618 : 1 | ±5% | -15 RS |
| Spire height : Base diameter | 1.618 : 1 | ±8% | -5 RS |
| Column spacing : Column height | 0.618 : 1 | ±5% | -10 RS |
| Window height : Width | 1.618 : 1 | ±10% | -3 RS |

### 4.3 Visual Feedback

- **Perfect proportions:** Building glows with golden aura, ornamental details auto-generate (cornices, moldings, rosettes appear as rewards)
- **Near-perfect (within tolerance):** Soft golden shimmer, basic ornamentation
- **Out of ratio:** No glow, flat textures, visible seam lines — the building looks *wrong* in a way that's hard to pinpoint but easy to feel

## 6. Precision Rock Cutting

### 6.1 Overview

Tartarians didn't chisel stone — they used **focused sound frequencies** to cut granite, marble, and basalt with laser precision. This mini-game captures that wonder.

### 6.2 Cutting Mechanics

**Giant Mode Required** (60-second bursts, upgradeable)

1. **Select stone block** from quarry/excavation
2. **Choose frequency** — different stones respond to different Hz:
   - Granite: 174 Hz (foundational)
   - Marble: 285 Hz (healing, smooth)
   - Basalt: 396 Hz (protective)
   - Crystal: 528 Hz (precise, brittle)
   - Obsidian: 639 Hz (sharp, resonant)
3. **Trace the cut line** — finger draws the golden-spiral cut path
4. **Sustain the note** — hold steady while the frequency parts the stone
5. **Quality rating** based on accuracy, consistency, and golden-ratio compliance

### 6.3 Cut Quality

| Rating | Visual | RS Bonus | Description |
|---|---|---|---|
| **Perfect** | Mirror-smooth surface, golden glow | +15 RS | "The stone sang as it split" |
| **Excellent** | Near-smooth, faint shimmer | +10 RS | Barely visible cut line |
| **Good** | Clean cut, visible seam | +5 RS | Functional and solid |
| **Rough** | Chipped edges, no shimmer | +0 RS | Works but ugly |

### 6.4 Advanced Cuts (Unlocked Moon 7+)

After Korath's mentorship:

| Cut Type | Technique | Use |
|---|---|---|
| **Spiral Cut** | Trace golden spiral through block | Ornamental columns |
| **Tongue & Groove** | Interlocking precision cut | Earthquake-proof walls |
| **Acoustic Chamber** | Hollow interior with resonant geometry | Dome interiors |
| **Megalith Shaping** | Shape 300-ton blocks for airship transport | Continental building |

## 10. Player Building Flow

### 10.1 First-Time Builder (Moon 1)

1. Find buried structure → excavate (swipe away mud)
2. System highlights golden-ratio guides → "Place this block where the gold line shows"
3. Block auto-snaps → satisfying click + glow
4. Place spire on dome → sparks climb → "Your first resonance!"
5. Player learns: beautiful = powerful. No separate tutorial needed.

### 10.2 Intermediate Builder (Moon 4–6)

1. Choose archetype (dome vs. fort vs. pavilion)
2. Select template (Seed of Life? Golden Rectangle?)
3. Precision cut materials (mini-game)
4. Place elements using snap system
5. Connect to grid (routing puzzle)
6. Tune and activate (organ/bell mini-game)
7. Watch it come alive — NPCs arrive, Aether flows, beauty intensifies

### 10.3 Master Builder (Moon 10+)

1. Design district-scale fractal layouts
2. Combine ALL archetypes into integrated complexes
3. Multi-zone ley-line corridors connecting to continental grid
4. Legendary materials + perfect golden-ratio compliance
5. 27× multiplier zones with floating gardens + perpetual auroras
6. NPC-driven festivals celebrating the player's architectural vision
